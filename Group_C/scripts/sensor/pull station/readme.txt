To run this folder
1. Install thonny
2. Select Tools>options>Interpreter>\
		code : Micropython(ESP32)
		Port : Silicon Labs ..COM3
	then upload firmware (esp32-20220618-v1.19.1.bin) given in this folder at
		Install or update Micropython

3. Select boot.py and save as ... to micropython device
4. Do step3 same for main.py and umqttsimple.py
5. Then run
